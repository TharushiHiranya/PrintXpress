# User stories

## Accounts and profile

- **US-01 (Must)** As a new customer, I want to register with my email or phone number and a password, so that I can save my orders and designs.
- **US-02 (Must)** As a returning customer, I want to log in with my email or phone and password, so that I can reach my account.
- **US-03 (Should)** As a customer, I want to view and edit my name, email, and phone, so that my details stay correct.
- **US-04 (Should)** As a customer, I want to save and edit delivery addresses, so that I can check out faster.
- **US-05 (Should)** As a customer, I want to log out, so that my account is safe on a shared phone.

## Browsing products

- **US-06 (Must)** As a customer, I want to see categories of print products, so that I can find what I need.
- **US-07 (Must)** As a customer, I want to open a product and see its price, specifications, and a sample preview, so that I can decide what to order.
- **US-08 (Could)** As a customer, I want to filter products by type, material, or size, so that I can narrow the list quickly.

## Designs and ordering

- **US-09 (Must)** As a customer, I want to upload my own design file, so that the shop prints exactly what I want.
- **US-10 (Must)** As a customer, I want to type custom text instead of a file, so that I can order simple jobs without a design.
- **US-11 (Must)** As a customer, I want to choose quantity, paper or material, and size, then place the order, so that my print job is requested.
- **US-12 (Could)** As a customer, I want to save a design to my account, so that I can reuse it on a later order.

## Managing orders

- **US-13 (Must)** As a customer, I want to see the status of each order, so that I know when it will be ready.
- **US-14 (Must)** As a customer, I want to cancel or reschedule an order before printing starts, so that I can fix mistakes.

## Notifications and offers

- **US-15 (Must)** As a customer, I want notifications when my order is confirmed and completed, so that I do not have to keep checking.
- **US-16 (Should)** As a customer, I want to see seasonal offers and discount codes, so that I can save money on bulk or festive orders.

## Help and support

- **US-17 (Should)** As a customer, I want guidance on file requirements, colour formats, and bleed margins, so that my print comes out right.
- **US-18 (Should)** As a customer, I want FAQs and a way to ask for help, so that I can solve design questions.
